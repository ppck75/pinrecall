# Pinrecall

Pinrecall은 시험 준비와 반복 학습을 위한 웹 학습 도구입니다. 외워야 할 내용은 포스트잇처럼 칠판에 붙이고, 단어장이나 오답노트는 퀴즈 카드로 만들어 테스트할 수 있으며, 표 기능으로 행과 열이 있는 학습 자료도 정리할 수 있습니다.

로그인한 계정별로 포스트잇, 칠판, 퀴즈가 Supabase에 저장되므로 여러 기기에서도 같은 학습 자료를 이어서 사용할 수 있습니다. 표 데이터는 로그인 사용자별 브라우저 저장소에 저장됩니다.

## 주요 기능

- `#notes`: 계정별 작업공간, 포스트잇 메모 보드, 여러 칠판
- `#quiz`: 문제/정답 등록, 정답 보기, 이전/다음 문제, 셔플, 20개 단위 퀴즈 목록 페이지
- `#table`: 20행 단위 표 페이지, 세로줄 추가, 셀 더블클릭 편집
- `#docs`: 핵심 사용 설명서
- `#signup`: 이메일, 비밀번호, 비밀번호 확인, 닉네임으로 회원가입
- `#login`: 로그인

## 사용 설명서

### 1. 포스트잇 보드

외워야 할 단어, 오답 이유, 자주 틀리는 문법을 짧게 적어 칠판에 붙입니다. 포스트잇은 드래그해서 원하는 위치로 옮길 수 있습니다.

### 2. 작업공간과 여러 칠판

작업공간을 나누고, 각 작업공간 안에서 `칠판 추가`를 눌러 여러 칠판을 만들어 학습 내용을 분리할 수 있습니다.

### 3. 퀴즈 카드

문제와 정답을 등록하고 `정답 보기`, `이전`, `다음`, `셔플`을 사용해 단어장이나 오답노트처럼 테스트합니다. 추가한 퀴즈 목록은 20개 단위로 페이지가 나뉩니다.

### 4. 표

메뉴에서 `표`를 열면 한 페이지당 20행으로 제한된 표를 사용할 수 있습니다. `세로줄 추가`로 열을 늘릴 수 있으며, 화면이 깨지지 않도록 최대 8칸까지 지원합니다. 셀을 더블클릭하면 텍스트를 입력할 수 있고, `페이지 추가`, `이전`, `다음`으로 여러 표 페이지를 오갈 수 있습니다.

### 5. 계정별 저장

로그인한 계정별로 포스트잇, 칠판, 퀴즈가 Supabase에 저장됩니다. 다른 계정의 자료와 섞이지 않습니다. 표 데이터는 현재 브라우저의 로그인 사용자별 저장소에 저장됩니다.

## 기술 스택

- HTML: 정적 페이지 구조
- CSS: 반응형 UI, 포스트잇 보드, 드롭다운 메뉴, 표 그리드, Pretendard 폰트
- JavaScript: 해시 라우팅, 작업공간 전환, 포스트잇 드래그, 퀴즈 셔플, 표 편집, Supabase API 연동
- Supabase Auth: 회원가입, 로그인, 세션 관리
- Supabase Database: 프로필, 작업공간, 칠판, 포스트잇, 퀴즈 데이터 저장
- Supabase RLS: 로그인한 사용자별 데이터 접근 제어
- Vercel: 정적 웹사이트 배포
