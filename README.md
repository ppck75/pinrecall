# Pinrecall

Pinrecall은 시험 준비와 반복 학습을 위한 웹 학습 도구입니다. 외워야 할 내용은 포스트잇처럼 칠판에 붙이고, 단어장이나 오답노트는 퀴즈 카드로 만들어 테스트할 수 있습니다.

로그인한 계정별로 포스트잇, 칠판, 퀴즈가 Supabase에 저장되므로 여러 기기에서도 같은 학습 자료를 이어서 사용할 수 있습니다.

## ✨ 주요 기능

- `#notes`: 계정별 작업공간, 포스트잇 메모 보드, 여러 칠판
- `#quiz`: 문제/정답 등록, 정답 보기, 다음 문제, 셔플
- `#docs`: 핵심 사용 설명서
- `#signup`: 이메일, 비밀번호, 비밀번호 확인, 닉네임으로 회원가입
- `#login`: 로그인

## 📘 사용 설명서

### 📝 1. 포스트잇 보드

외워야 할 단어, 오답 이유, 자주 틀리는 문법을 짧게 적어 칠판에 붙입니다. 포스트잇은 드래그해서 원하는 위치로 옮길 수 있습니다.

### 🧭 2. 작업공간과 여러 칠판

작업공간을 토익, 학교 메모처럼 나누고 각 작업공간 안에서 `칠판 추가`를 눌러 칠판을 넘기며 정리합니다.

### ✅ 3. 퀴즈 카드

문제와 정답을 등록한 뒤 `정답 보기`, `다음`, `셔플`을 사용해 단어장이나 오답노트처럼 테스트합니다.

### 🔐 4. 계정별 저장

로그인한 계정별로 포스트잇, 칠판, 퀴즈가 Supabase에 저장됩니다. 다른 계정의 자료와 섞이지 않습니다.

## 🛠 기술 스택

- HTML: 정적 페이지 구조
- CSS: 반응형 UI, 포스트잇 보드, 드롭다운 메뉴, Pretendard 폰트
- JavaScript: 해시 라우팅, 작업공간 전환, 포스트잇 드래그, 퀴즈 셔플, Supabase API 연동
- Supabase Auth: 회원가입, 로그인, 세션 관리
- Supabase Database: 프로필, 작업공간, 칠판, 포스트잇, 퀴즈 데이터 저장
- Supabase RLS: 로그인한 사용자별 데이터 접근 제어
- Vercel: 정적 웹사이트 배포
